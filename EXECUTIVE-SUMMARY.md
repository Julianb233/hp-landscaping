## üìä Performance Targets

### Core Web Vitals (Critical for SEO)
- **LCP** (Largest Contentful Paint): < 2.5 seconds
- **FID** (First Input Delay): < 100 milliseconds
- **CLS** (Cumulative Layout Shift): < 0.1

### Lighthouse Scores (Target: 90+)
- **Performance**: 90+
- **Accessibility**: 95+
- **Best Practices**: 95+
- **SEO**: 95+

### Why This Matters
- Google uses Core Web Vitals as ranking factor
- Fast sites convert better (53% of mobile users abandon sites that take > 3s)
- Accessibility compliance reduces legal risk
- Higher scores = better user experience = more leads

## üéØ SEO Strategy

### On-Page SEO
- Semantic HTML5 markup
- Proper heading hierarchy
- Meta titles/descriptions on every page
- Open Graph tags for social sharing
- Alt text for all images
- Internal linking between related pages
- Keyword optimization

### Technical SEO
- XML sitemap (auto-generated)
- Robots.txt
- Canonical URLs
- Structured data (LocalBusiness, Service, Review schemas)
- Mobile-friendly responsive design
- HTTPS (SSL certificate)

### Local SEO (Critical for Landscaping Business)
- Google Business Profile optimization
- NAP consistency (Name, Address, Phone) everywhere
- Local keywords in content
- Service area pages
- Location-based schema markup
- Embedded Google Maps
- Local citations (Yelp, Angi, HomeAdvisor)

### Content SEO
- Blog with seasonal articles (spring prep, fall cleanup, etc.)
- Regular updates
- Long-tail keywords ("landscape design in [city]")
- FAQ content
- Local plant guides
